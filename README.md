# HarmonyBook - 淘小说书城

## 简介
HarmonyBook 是一个基于鸿蒙技术开发的淘小说书城应用，旨在为用户提供一个便捷的在线阅读平台。用户可以在此平台上浏览、搜索和阅读各种小说。

## 特性
- **用户友好的界面**：简洁直观的设计，提升用户体验。
- **丰富的小说资源**：汇聚了大量热门小说，满足不同读者的需求。
- **章节目录功能**：方便用户快速找到所需章节。
- **书城功能**：提供书籍推荐和分类浏览。

## 安装
使用以下命令克隆并安装项目：

```bash
git clone https://github.com/brave-jee/HarmonyBook.git
cd HarmonyBook
npm install
```

## 使用

在安装完成后，可以运行以下命令启动应用：

```
npm start
```
